      • D : 自动位数日（1-31）

      • h : 自动位数时（十二小时制）（1-12）
      • mm : 两位数分（00-59）
      • m : 自动位数分（0-59）
      • ss : 两位数秒（00-59）
      • s : 自动位数秒（0-59）
      • f : 时分秒
      • ff : 百分秒
      • fff : 千分秒
      • tt : 上午/下午（小写）
      • t : 晨/昏（小写）
      • TT : 上午/下午
      • T : 晨/昏
      • zzzz: 四位时区（例如-05：00）
      • zzz : 三位时区（例如-5：00）
      • zz : 两位时区（例如-05）
      • z : 一位时区（例如-5）
