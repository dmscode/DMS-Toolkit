dms-userscript-toolkit

稻米鼠的工具类

把自己日常使用的一些操作写成类，方便复用。同时学习 TypeScript 和 Node.js 下一些开发工具的使用。

目前设定编译目标为：ES3，因为我有在一些老旧设备中使用的需求。如果有其他需要应该完全可以编译至更高版本。

详细使用方法见 ./docs
